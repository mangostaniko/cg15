Detailed Description

A SceneObject holds a transformation matrix and its inverse and provides functions to manipulate them. This is intended as a base class for all objects that appear in the scene.

Member Enumeration Documentation

enum SceneObject::Order

enum specifying matrix multiplication order

Enumerator
LEFT 

incoming matrix is positioned left of current matrix in multiplication: M_result = M_inc * M.

RIGHT 

incoming matrix is positioned right of current matrix in multiplication: M_result = M * M_inc.
